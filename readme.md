# Crypto Converter

A simple web application built using HTML, CSS, and JavaScript.

## Features

- User Login Authentication
- Fetch users from users.json
- Crypto to Currency Conversion
- Live CoinGecko API
- Save Favorite Pairs
- Logout Functionality

## Technologies Used

- HTML
- CSS
- JavaScript
- CoinGecko API
- Git
- GitHub
